# VoyageWise
Smart Voyage is an AI-powered travel planning platform that helps travelers discover destinations, build personalized itineraries, estimate travel budgets, and organize every aspect of their trips in one place.  Designed with usability and automation in mind, Smart Voyage combines modern web technologies with artificial intelligence. Using Base44, demonstrating skills in prompt engineering, AI-assisted development, application design, workflow automation, debugging, and rapid prototyping to deliver a functional end-to-end solution.
